新电脑系统配置[个人向]

发表于2026-01-01|更新于2026-01-01
|浏览量:|评论数:

正好今天重装了系统，把重装后的系统重新设置了一下。正好写一篇重装系统后需要做哪些工作。

系统设置

  • 关闭任务栏搜索框（鸡肋功能）

  • 解除（锁定任务栏）

  • 打开网络共享

  • 关闭快速访问

  • 显示文件扩展名

    安装软件

  • 微PE工具箱

  • 安装运行库

  • 搜狗输入法精简版

  • Chrome浏览器

  • Firefox浏览器

  • 360驱动大师

  • 利用Office Tool Plus安装Office

  • 解压缩软件7zip

  • 图片浏览器Honeyview

  • 视频播放器Potplayer

  • 稻壳阅读器

  • MAS激活Windows和Office等安装好Office后再一块激活Windows和Office就行。

游戏向

  • Steam
  • 雷神加速器

编程向

  • Git
  • Node.js
  • VS Code
  • PyCharm
  • 安装 pyenv-win
